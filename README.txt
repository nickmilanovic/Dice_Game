Dice Game built using class components in react.

Click on the number of dice to add one to the board.
Click on each dice to increment the number of sides.
Click the up arrow to increase the number of sides of all dice up to 20.
Click the down arrow to decrease the number of sides of all dice down to 1.
Click roll to randomize each dice between 1 and 20.
Click reset to delete all dice except 1 which will display a 1.
